# Experiment Template

## 目标

本文提供通信实验记录模板，用于沉淀接线、参数、波形、日志、问题和结论。

## 基本信息

```text
实验名称：
日期：
负责人：
协议/接口：
主控平台：
目标设备：
软件版本：
硬件版本：
```

## 实验目标

说明本次实验要验证什么：

```text
读取设备 ID
验证最大通信速率
验证错误恢复
验证 DMA 接收
验证长线稳定性
```

## 硬件连接

记录接线：

```text
信号名 | 主控引脚 | 目标引脚 | 备注
```

记录关键硬件条件：

- 电压域。
- 上拉/下拉。
- 终端电阻。
- 收发器型号。
- 线缆长度。
- 是否隔离。

## 参数配置

记录协议参数：

```text
UART：baud/data/parity/stop
I2C：speed/address/pull-up
SPI：mode/frequency/bit order/CS
CAN：bitrate/sample point/frame type
Ethernet：IP/mask/gateway/port
USB：VID/PID/class/endpoints
```

## 测试步骤

1. 上电检查。
2. 初始化接口。
3. 执行最小通信。
4. 读取固定 ID 或状态。
5. 执行目标实验动作。
6. 记录结果。
7. 执行异常或边界测试。

## 观测记录

记录：

- 示波器截图。
- 逻辑分析仪解码。
- Wireshark 抓包。
- CAN/USB/PCIe 工具日志。
- 串口日志。
- 错误计数。
