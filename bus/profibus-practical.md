# PROFIBUS Practical Guide

## 目标

本文用于 PROFIBUS 工程调试，重点覆盖 DP/PA 区分、地址、终端、GSD、主站组态、从站诊断和存量系统排查。

## 先分清类型

```text
PROFIBUS DP：分布式 IO，工厂自动化常见
PROFIBUS PA：过程仪表，过程自动化常见
```

工程中最常见调试对象是 PROFIBUS DP。

## 最小检查

- 主站和从站类型匹配。
- 从站地址唯一。
- GSD 文件正确。
- 总线终端正确。
- 波特率和组态一致。
- 设备供电正常。

## GSD

GSD 是设备描述文件。

它描述：

- 设备能力。
- IO 模块。
- 参数。
- 诊断信息。
- 支持波特率。

PLC 工程组态必须使用正确版本 GSD。

## 终端和布线

PROFIBUS DP 基于 RS485 物理层时，终端和布线非常关键。

要点：

- 总线两端终端开启。
- 中间节点终端关闭。
- 分支尽量短。
- 使用合适 PROFIBUS 电缆和连接器。

## 调试步骤

1. 确认主站工程中设备型号和 GSD。
2. 设置从站地址。
3. 检查终端电阻。
4. 单从站接入测试。
5. 查看主站诊断。
6. 逐个增加从站。
7. 记录错误站点和诊断码。

## 常见故障

| 现象 | 优先检查 |
| --- | --- |
| 主站找不到从站 | 地址、GSD、接线、终端 |
| 多站后不稳定 | 终端、线缆、分支、屏蔽接地 |
| IO 长度不匹配 | 组态模块、GSD 版本 |
| 只某站异常 | 地址重复、设备故障、连接器 |

## 检查清单

- 地址唯一。
- GSD 与设备匹配。
- 两端终端正确。
- 总线线缆和连接器符合规范。
- 主站诊断信息已记录。
