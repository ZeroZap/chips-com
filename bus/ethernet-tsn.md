# Ethernet TSN

## 定位

TSN 是 Time-Sensitive Networking，时间敏感网络，是一组以太网实时增强标准。

它用于在标准以太网生态中提供确定性传输能力。

## 目标

TSN 试图解决普通 Ethernet 的不确定延迟问题。

典型能力：

- 时间同步。
- 流量调度。
- 带宽预留。
- 低抖动传输。
- 冗余和可靠性。

## 常见场景

- 工业自动化。
- 车载以太网。
- 音视频实时传输。
- 分布式控制。

## 常见标准方向

入门只需知道 TSN 是一组标准，不是单一协议。

常见方向：

- 时间同步。
- 时间感知调度。
- 帧抢占。
- 流过滤和监管。
- 冗余传输。

## 与 EtherCAT 的关系

EtherCAT 是专门的实时工业以太网协议。

TSN 是标准以太网增强方向。

未来工业以太网和车载以太网可能更多利用 TSN 能力。

## 工程注意

- 需要 TSN-capable MAC/交换机。
- 需要系统级时间同步。
- 配置复杂度高。
- 协议栈和生态仍需按项目确认。
