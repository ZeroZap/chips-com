# PWM

## 定位

PWM 是脉宽调制，用数字输出模拟平均功率或控制时序。

常见用途：

- LED 调光。
- 蜂鸣器。
- 电机调速。
- 舵机控制。
- 背光控制。
- 简单 DAC 滤波输出。

## 核心参数

```text
Frequency：频率
Period：周期
Duty Cycle：占空比
Resolution：分辨率
Polarity：极性
```

占空比：

```text
高电平时间 / 总周期
```

## 频率和分辨率

频率越高，周期越短。

在定时器时钟固定时，频率越高，可用计数范围越小，分辨率越低。

选型要平衡：

```text
PWM 频率
占空比分辨率
负载响应
EMI
```

## 舵机 PWM

常见舵机控制：

```text
周期约 20 ms
脉宽约 1 ms ~ 2 ms
```

具体以舵机手册为准。

## LED 调光

LED PWM 频率太低会闪烁。

频率太高会增加开关损耗或降低分辨率。

背光调光还要注意：

- 人眼闪烁。
- 摄像头频闪。
- 最小占空比。
- 极性。

## 常见错误

| 现象 | 优先检查 |
| --- | --- |
| 没波形 | 定时器时钟、引脚复用、输出使能 |
| 频率不对 | 预分频、周期寄存器、时钟源 |
| 占空比反了 | 极性配置 |
| 抖动 | 时钟、更新寄存器、软件频繁改值 |

## 检查清单

- 定时器时钟确认。
- 频率和周期计算正确。
- 占空比边界处理正确。
- 引脚复用为 PWM 输出。
- 极性和负载匹配。
