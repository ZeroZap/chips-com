# GPIO

## 定位

GPIO 是通用输入输出口，是所有芯片外设通信和控制信号的基础。

常见用途：

- LED 控制。
- 按键输入。
- 复位脚控制。
- 中断输入。
- 片选 CS。
- 电源使能。
- 模拟协议时序。

## 输入和输出

输入模式：

```text
读取外部高低电平
```

输出模式：

```text
主动驱动高低电平
```

## 上拉和下拉

输入引脚不能浮空。

常见配置：

```text
Pull-up：默认高电平
Pull-down：默认低电平
Floating：浮空，通常不推荐
```

按键输入常用上拉或下拉确定默认状态。

## 推挽和开漏

推挽输出：

```text
可以主动拉高和拉低
速度快，驱动能力强
```

开漏输出：

```text
只能主动拉低
高电平靠上拉电阻
适合共享线和线与逻辑
```

I2C 就是典型开漏总线。

## 中断

GPIO 可配置中断触发：

```text
上升沿
下降沿
双边沿
高电平
低电平
```

按键、传感器 INT、触摸中断都常用 GPIO 中断。

## 去抖

机械按键会抖动。

常见处理：

- 软件延时确认。
- 定时采样。
- 状态机。
- RC 硬件滤波。

## 常见错误

| 现象 | 优先检查 |
| --- | --- |
| 输入乱跳 | 浮空、上拉/下拉缺失、干扰 |
| 输出不动 | 引脚复用、方向配置、外部短路 |
| 中断频繁触发 | 抖动、边沿配置、噪声 |
| 电平不对 | 电压域、开漏缺上拉、驱动能力 |

## 检查清单

- 引脚复用正确。
- 输入有确定默认电平。
- 输出模式匹配推挽/开漏需求。
- 电压域兼容。
- 中断触发方式正确。
- 按键或慢信号有去抖。
