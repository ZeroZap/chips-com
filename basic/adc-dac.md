# ADC / DAC

## 定位

ADC 把模拟电压转换为数字值。

DAC 把数字值转换为模拟电压。

常见用途：

- 电池电压采样。
- 温度、电流、压力传感器。
- 音频和波形输出。
- 控制参考电压。

## ADC 核心参数

```text
Resolution：分辨率
Reference Voltage：参考电压
Sampling Rate：采样率
Input Range：输入范围
Input Impedance：输入阻抗
```

N 位 ADC 数值范围：

```text
0 ~ 2^N - 1
```

电压换算：

```text
Voltage = Raw / (2^N - 1) * Vref
```

## 采样注意

ADC 采样受影响：

- 参考电压精度。
- 输入源阻抗。
- 采样时间。
- PCB 噪声。
- 电源纹波。
- 校准误差。

高阻信号需要更长采样时间或运放缓冲。

## DAC 核心参数

DAC 关注：

- 分辨率。
- 输出范围。
- 输出驱动能力。
- 建立时间。
- 噪声。
- 参考电压。

DAC 输出不一定能直接驱动大负载。

## 常见错误

| 现象 | 优先检查 |
| --- | --- |
| ADC 值飘 | 参考电压、输入阻抗、噪声、采样时间 |
| ADC 饱和 | 输入超范围、分压错误 |
| 读数偏差大 | 校准、Vref、分压电阻精度 |
| DAC 输出不准 | 负载过重、参考电压、缓冲不足 |

## 检查清单

- 输入电压不超过范围。
- Vref 稳定且已参与换算。
- 采样时间匹配源阻抗。
- 模拟地和数字噪声处理合理。
- 需要时做校准和滤波。
